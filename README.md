**原文标题**: I'm helping my dog vibe code games

**原文链接**: [https://www.calebleak.com/posts/dog-game/](https://www.calebleak.com/posts/dog-game/)

在被 Meta 裁员后，一位研究工程师兼作家开发了一套系统，让他 9 磅重的卡瓦普犬 Momo 利用 Claude Code 进行“氛围感编程”（vibe code）来制作视频游戏。该项目起源于 Momo 意外踩踏了作者的键盘，从而引发了一个思考：AI 是否能将犬类的“胡言乱语”解读为有意义的设计。

技术流程包括 Momo 在蓝牙键盘上敲击，信号通过树莓派代理发送至名为“DogKeyboard”的自定义 Rust 应用程序。该程序负责过滤输入并将其转发给 Claude Code。为了实现自动化，系统还通过 Zigbee 集成了一台智能宠物喂食器，一旦 Momo 提供了足够的输入，便会获得零食奖励。

为了弥合随机按键与功能性软件之间的差距，作者使用了一个特定的提示词，指示 Claude 扮演一名才华横溢的开发人员，负责解读来自一位“古怪设计师”的“神秘指令”。他选择了 Godot 引擎进行开发，因为该引擎基于文本的场景文件易于被 AI 操作。为确保游戏的可玩性，作者实现了自动化反馈循环，包括用于抓取屏幕截图的 Python 脚本，以及一个输入测试工具，让 Claude 能够通过运行自己创建的游戏来充当自身的 QA 测试员。

经过两周的正向激励训练，Momo 学会了通过刷键盘来触发 AI。实验最终产出了几款功能完备的 3D 游戏——例如《沼泽点心》（Swamp Snacker）和《八角律动》（Octogroove）——且完全由 C# 编写。作者总结道，虽然狗狗提供了“氛围感”（vibes），但项目的成功仍依赖于强大的约束机制、专门的代码检查器以及 AI 通过视觉和逻辑反馈进行自我纠错的能力。

## 4. 三星升级再造承诺

**原文标题**: Samsung Upcycle Promise

**原文链接**: [https://www.xda-developers.com/samsung-promised-make-old-phones-useful-galaxy-upcycle/](https://www.xda-developers.com/samsung-promised-make-old-phones-useful-galaxy-upcycle/)

2017年，三星宣布了一项具有革命意义的“Galaxy Upcycling”（Galaxy 升级再造）计划，承诺通过解锁引导加载程序（bootloader）并为自定义应用提供开源市场，帮助用户重新利用旧智能手机。通过与 iFixit 合作，该愿景旨在将旧硬件转化为从比特币矿机到智能家居控制枢纽的各种设备。

然而，该计划随后陷入了多年的沉寂并面临内部阻力。批评者认为，该项目缺乏盈利模式，且与三星销售新硬件的核心业务直接冲突。当“Galaxy Upcycling at Home”最终于 2021 年推出时，它已远非最初承诺的模样。它并未提供开源生态系统，而仅提供了两项基本功能——声音和光线感应，且仅限于仍具有较高转售价值的较新型号设备。

尽管这种实施方案大打折扣，三星仍凭借该项目获得了路透社颁发的“循环转型奖”。作者强调了这一奖项的讽刺性，指出三星实际的电子垃圾回收率依然微乎其微（0.0019%）。此外，iFixit 于 2024 年正式结束了与三星的合作，理由是零部件价格高昂、维修政策苛刻，以及产品设计优先考虑大量使用胶水而非易维修性。

文章最后总结道，虽然用户仍可以使用第三方应用或自定义 ROM（如 LineageOS）自行开展升级再造项目，但三星官方提出的由厂商支持的开源生态系统愿景实际上已经破灭。该公司选择优先考虑硬件销售和软件控制，而非真正的可持续发展，这使得最初的“升级再造”承诺最终沦为一场成功的公关秀。

自1994年启用以来，位于大阪的关西国际机场始终保持着从未丢失过一件行李的惊人纪录。这一长达30年的行李处理零失误表现，使其在2024年世界机场大奖（World Airport Awards 2024）中荣膺“世界最佳行李交付机场”称号——这也是该机场第八次获此殊荣。

该机场的成功归功于先进技术与细致人工监管的结合。在高峰时段，机场每天处理多达3万件托运行李。为防止遗失，机场在传送带沿线安装了传感器，并辅以工作人员在货物区域巡逻，以检查是否有掉落的物品。

关西机场也因其卓越的客户服务而闻名。地勤人员会手动调整行李提取转盘上行李箱的拉手，使其统一朝外，以便乘客更轻松地提取行李。

展望未来，该机场面临着人手短缺以及2025年大阪关西世博会预期客流激增的双重挑战。为保持其完美纪录并维持高标准服务，机场官员计划投资进行系统升级和技术改进，以提高运营效率。

## 6. 致谷歌关于强制应用分发开发者注册的公开信

**原文标题**: Open Letter to Google on Mandatory Developer Registration for App Distribution

**原文链接**: [https://keepandroidopen.org/open-letter/](https://keepandroidopen.org/open-letter/)

包括 EFF、F-Droid 和 Proton 在内的 37 家公民社会组织和科技公司组成的联盟，向谷歌领导层发出公开信，反对一项要求所有安卓应用分发必须进行强制性开发者注册的新政策。该政策计划于 2026 年初生效，规定即使是通过第三方商店或直接下载分发应用的开发者，也必须向谷歌注册、提供政府身份证件并支付费用。

签署方认为，此举将谷歌定位为“全球守门人”，从根本上削弱了安卓系统长期以来的开放性。信中列出的主要担忧包括：

*   **准入门槛：** 该政策制造了行政和财务障碍，对开源项目、小型开发者和人道主义组织造成了不成比例的影响。
*   **隐私与监控：** 它建立了一个全球开发者的中心化数据库，为从事政治敏感或隐私保护工具开发的人员带来了风险。
*   **反竞争优势：** 该强制令允许谷歌监控其市场之外的所有开发活动，从而获得不公平的情报优势并引发反垄断担忧。
*   **专断控制：** 注册中心化赋予了谷歌单方面禁用整个生态系统内应用程序的权力，无论其分发渠道如何。

公开信指出，安卓现有的安全功能（如沙箱机制和用户警告）已足以保护用户。该联盟呼吁谷歌立即撤回该政策并承诺保持平台中立。他们总结道，集中审批所有软件的权力是对数字主权、公平竞争以及安卓生态系统赖以建立的言论自由原则的威胁。

**原文标题**: Stripe valued at $159B, 2025 annual letter

**原文链接**: [https://stripe.com/newsroom/news/stripe-2025-update](https://stripe.com/newsroom/news/stripe-2025-update)

Stripe 2025年年度信函揭示了公司过去一年显著的财务增长，以及向AI驱动的“代理商业”（Agentic Commerce）的战略转型。在Thrive Capital和a16z等投资者的支持下，Stripe通过旨在为员工提供流动性的要约收购，公司估值达到了**1590亿美元**。

**财务表现与市场覆盖**
Stripe处理的总交易额达到**1.9万亿美元**（较2024年增长34%），相当于全球GDP的1.6%。公司保持强劲盈利，其非支付收入套件的年化营收接近10亿美元。Stripe在大型企业中的市场主导地位已近乎普及，为90%的道琼斯指数成份股公司和80%的纳斯达克100指数成份股公司提供服务。

**“默认全球化”的转型**
2025届新入驻企业是Stripe历史上表现最强劲的群体，这些公司达到1000万美元年度经常性收入（ARR）的速度是去年的两倍。值得注意的是，受“默认全球化”模式推动——特别是在OpenAI和Midjourney等AI初创公司中——57%的新业务源自美国以外。

**AI与代理商业**
Stripe正将自己定位为AI经济的金融基础设施层。关键进展包括：
*   **代理商业协议 (ACP)：** 与OpenAI共同开发，旨在允许AI代理执行交易。
*   **代理商业套件：** 为Etsy和Coach等品牌提供工具，使其能够通过AI接口进行销售。
*   **共享支付令牌：** 一种新的技术原语，允许AI代理在不泄露敏感凭证的情况下完成支付。

**稳定币与区块链**
尽管比特币市场波动，Stripe上的稳定币交易额仍翻倍至4000亿美元。为此，Stripe收购了**Bridge**和**Privy**，并推出了专门为支付设计的区块链**Tempo**。这些举措旨在为蓬勃发展的“代币经济”提供亚秒级的结算确认和专用支付通道。
