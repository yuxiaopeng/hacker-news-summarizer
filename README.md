**能力突破：**
*   **编程与科学：** Sol 在 Artificial Analysis 编程智能体指数（Coding Agent Index）和“智能体终极考试”（Agents’ Last Exam）中刷新了纪录，在专业工作流和网络安全领域展现出卓越的推理能力（例如，在 ExploitGym 中的表现比 GPT-5.5 翻了一番）。
*   **设计与知识工作：** 该模型展现了先进的“设计判断力”，能够创建并优化功能性用户界面和专业演示文稿。它通过与 Microsoft 365、Slack 和 Notion 等企业级工具集成，将原始数据转化为精美、可共享的成品。
*   **计算机使用：** 经过改进的自主“计算机使用”能力使模型能够检查、调试和优化渲染结果，而不仅仅是生成代码。

**安全性：**
GPT-5.6 包含了 OpenAI 迄今为止最强大的安全防护措施，通过广泛的人工红队测试和自动化测试开发而成，在保持专业工作高实用性的同时严防滥用。

在经历了与 Microsoft、Notion 和 Canva 等合作伙伴的有限预览期后，GPT-5.6 系列现已全面开放。

## 2. ChatGPT Work

**原文标题**: ChatGPT Work

**原文链接**: [https://openai.com/index/chatgpt-for-your-most-ambitious-work/](https://openai.com/index/chatgpt-for-your-most-ambitious-work/)

OpenAI has announced **ChatGPT Work**, an AI agent powered by the new **GPT-5.6** frontier model, designed to execute complex, multi-step workflows across web, mobile, and desktop platforms. Moving beyond simple conversation, ChatGPT Work can independently manage projects for hours, breaking down goals into finished materials like documents, slides, and web apps.

**Key Features:**
*   **Action-Oriented Workflows:** Integration with Codex technology allows the agent to navigate apps, use a built-in browser, and interact with local files. A new "Computer Use" capability enables the desktop app to click, type, and move files on behalf of the user.
*   **Automation:** "Scheduled Tasks" allow users to automate repetitive actions, such as turning Slack messages into project updates or monitoring dashboards for changes.
*   **Custom Environments:** The "Sites" feature allows users to convert ideas into interactive web apps or live dashboards. 
*   **Integration:** Through an updated plugin directory, ChatGPT connects to tools like Microsoft 365, Google Workspace, Slack, and various CRMs.

**Enterprise & Security:**
Early adopters, including NVIDIA and Zapier, have used the tool to automate lead reviews and large-scale event analysis. To ensure safety, OpenAI introduced "Auto-review," an AI-driven security layer that blocked 100% of data extraction attempts during testing. Admins maintain granular control over tool access and spending limits.

**Availability:**
ChatGPT Work is rolling out now for Pro, Enterprise, and Edu users, with Plus and Business plans to follow. The updated desktop app—which merges with the Codex app—is available globally for all users, including those on the Free plan. Existing desktop installations will be renamed "ChatGPT Classic."
