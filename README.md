## 4. 法线贴图背后的几何学

**原文标题**: The Geometry Behind Normal Maps

**原文链接**: [https://www.shlom.dev/articles/geometry-behind-normal-maps/](https://www.shlom.dev/articles/geometry-behind-normal-maps/)

本文旨在揭秘法线贴图中的切线空间，阐释它并非一种渲染技巧，而是源于表面参数化（UV）的一种基本几何结构。切线空间是构建于表面每个点的局部坐标系，由切平面和法向量定义。UV坐标决定了这个坐标系的朝向，连接了平坦的纹理世界和弯曲的3D表面。

文章详细介绍了如何通过UV坐标和表面几何计算切向量和副切向量来构建切线空间，有效地找到将纹理空间方向映射到相应表面方向的变换。格拉姆-施密特正交化过程对于正交化切线坐标系至关重要。文章随后解释了法线贴图如何利用切线空间，通过将表面朝向存储为颜色，为低多边形网格添加精细细节。法线贴图像素存储切线空间中的方向，而切线基（TBN）矩阵将这些方向转换为表面的坐标系，从而影响光照计算。切线基的正确生成和插值对于精确的法线贴图至关重要，可以防止接缝和高光伪影。文章最后强调，切线空间是表面的一个基本方面，而不仅仅是一种着色技术，它支撑着各项技术，例如各向异性和置换贴图。

**原文标题**: Ioannis Yannas invented artificial skin for treatment of burns–dies at 90

**原文链接**: [https://news.mit.edu/2025/professor-ioannis-yannas-dies-1027](https://news.mit.edu/2025/professor-ioannis-yannas-dies-1027)

长期担任麻省理工学院教职的物理化学家伊奥尼斯·扬纳斯于10月19日去世，享年90岁。他最著名的成就是发明了用于治疗严重烧伤的人工皮肤，这一发现彻底改变了烧伤护理，并开创了再生医学领域。扬纳斯与马萨诸塞州总医院的约翰·伯克合作，开发了Integra，一种获得专利的人工皮肤，由覆盖在源自牛肌腱和鲨鱼软骨的胶原支架上的硅酮外层制成。这项发明可以防止感染和脱水，从而使健康的皮肤细胞再生。

在整个职业生涯中，扬纳斯获得了无数荣誉，包括当选为国家医学院院士、国家发明家名人堂院士和国家工程院院士。他撰写了具有影响力的著作，包括《成人组织和器官再生》。他的遗产包括拯救了无数烧伤患者的生命，并激励了下一代生物工程师。追悼会将在稍后公布。他留下了两个孩子，塔尼亚·克鲁扎克和阿列克谢·扬纳斯，以及五个孙辈。

2025年11月，Micro.blog推出了新的“Studio”层级，将视频托管引入其平台，定位为YouTube的独立替代方案。创始人Manton Reece旨在赋能用户拥有其视频内容和在网络上的存在，将Micro.blog现有的数据所有权理念扩展到视频领域。

作者强调，Micro.blog之前已经提供视频上传功能，但Studio使其成为一流的功能，具有流畅的播放和自动分辨率转换。Reece表示，Studio层级的目标是首先服务用户，而不是算法。

## 8. Transmeta，最后一家大型网络公司首次公开募股，后来怎么样了？

**原文标题**: What happened to Transmeta, the last big dotcom IPO

**原文链接**: [https://dfarq.homeip.net/what-happened-to-transmeta-the-last-big-dotcom-ipo/](https://dfarq.homeip.net/what-happened-to-transmeta-the-last-big-dotcom-ipo/)

戴夫·法夸尔的博文探讨了全美达（Transmeta）的兴衰，这是一家CPU公司，于2000年11月进行了最后一次主要的互联网泡沫时代IPO。虽然并非严格意义上的互联网公司，但其IPO标志着一个转折点，投资者对互联网公司的兴趣降温，为更传统的科技股铺平了道路。

全美达以其独特的CPU设计而著称，该设计使用翻译层在更高效的内核上执行x86指令。这使得他们有可能模拟不同的架构，但他们专注于流行的x86架构。Linus Torvalds曾在全美达工作，这增加了它的重要性。

尽管发布了两条CPU产品线（Crusoe和Efficeon），全美达在性能和速度方面难以与英特尔和AMD竞争。Crusoe落后于奔腾III的速度，而Efficeon无法跟上奔腾4或AMD Athlon 64等64位处理器的兴起。因此，全美达的CPU在低功耗设备中的应用有限。

最终，全美达未能蓬勃发展。Torvalds于2003年离职，2005年，该公司转向授权其知识产权。2009年，全美达被Novafora收购，后者随后将专利组合出售给Intellectual Ventures。Novafora在此之后不久停止运营。Intellectual Ventures现在授权全美达的专利，使这家公司成为遥远的回忆。全美达的硬件现在是一种稀有的收藏品，尽管缺乏广泛的怀旧感。文章总结说，全美达的命运更像网景或VA Linux，而不是红帽。
