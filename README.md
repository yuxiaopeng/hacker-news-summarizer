## 2. Mortality associated with non-optimal ambient temperatures from 2000 to 2019

**原文标题**: Mortality associated with non-optimal ambient temperatures from 2000 to 2019

**原文链接**: [https://www.researchgate.net/publication/353058947_Global_regional_and_national_burden_of_mortality_associated_with_non-optimal_ambient_temperatures_from_2000_to_2019_a_three-stage_modelling_study](https://www.researchgate.net/publication/353058947_Global_regional_and_national_burden_of_mortality_associated_with_non-optimal_ambient_temperatures_from_2000_to_2019_a_three-stage_modelling_study)

这项发表在《柳叶刀-星球健康》（*The Lancet Planetary Health*）上的研究，对2000年至2019年间与非适宜环境温度相关的死亡率进行了全面的全球评估。研究人员采用三阶段建模方法，利用来自43个国家750个地点的数据，估算了归因于寒冷和炎热的死亡负担。

**主要发现：**
*   **全球影响：** 全球约9.43%的死亡（每年约508万例）与非适宜温度有关。这相当于每10万居民中有74例超额死亡。
*   **寒冷与炎热：** 绝大多数与温度相关的死亡与寒冷有关。寒冷相关死亡占全球死亡人数的8.52%，而炎热相关死亡占0.91%。
*   **地区差异：** 超过半数的死亡发生在亚洲，尤其是东亚和南亚。然而，撒哈拉以南非洲和东欧的地区超额死亡率最高。
*   **时间趋势：** 2000年至2019年间，全球与非适宜温度相关的死亡率略有下降。具体而言，寒冷相关死亡率下降了0.51%，而炎热相关死亡率上升了0.21%。

**结论：**
该研究强调，虽然目前全球范围内寒冷气温导致的死亡人数较多，但炎热相关死亡率的上升趋势反映了气候变化日益严峻的影响。研究人员指出，尽管由于寒冷相关死亡的减少，过去二十年间的净死亡负担略有下降，但随着全球变暖的持续，这一趋势可能会发生逆转。研究结果凸显了采取针对性公共卫生干预和气候减缓策略的紧迫性，以应对极端及非适宜温度所带来的不断演变的风险。

## 3. 因为它以言语诉说

**原文标题**: Because It Speaks in Words

**原文链接**: [https://brianschrader.com/archive/because-it-speaks-in-words/](https://brianschrader.com/archive/because-it-speaks-in-words/)

《因其以言语诉说》

在《因其以言语诉说》一文中，作者探讨了人类与叙事之间深厚的情感联系，以及大语言模型（LLMs）如何利用这一基本的生物特性。尽管数学可能是“大自然的语言”，但人类是通过叙事来感知世界的。进化的本能使我们倾向于相信与我们交流的对象，这使得言语成为人类社会中最为强大的力量。

文章指出，大语言模型之所以能引发公众的广泛想象——而此前的机器学习技术均未能做到这一点——正是因为它们以言语而非数字进行交流。由于这些模型模仿人类的谈吐，我们会下意识地赋予其智能与权威。然而，作者警告说，大语言模型仅仅是一个没有思想、道德或生命体验的“预测引擎”或“鹦鹉”。它生成的文本在统计学上看似合理，却缺乏任何客观事实的基础。

作者将人工智能生成的内容比作“空热量”或高度加工的垃圾食品，认为技术正在劫持我们对故事和情感联系的先天需求。正如保质期长的零食只能充饥却无法提供营养，大语言模型提供了令人愉悦的“甜美”真相，却无法满足人类的心灵。这些模型的功能就像一个“诡计多端的宫廷谋士”，专门挑我们想听的话说，可能将我们引入空洞沟通的“恐怖谷”。

最终，作者断言，追寻真理依然是人类独有的“奥德赛”。尽管人工智能可以预测下一个词，但它无法体验世界，也无法建立真实的联系。人类依然是唯一能够学习和分享真实真理的生命，我们必须警惕机器那如塞壬般诱人的声音，它们以牺牲实质内容为代价，换取了表面的便捷。

## 4. 大多数争论关乎自我，而非观点。

**原文标题**: Most arguments are about ego, not ideas

**原文链接**: [https://wangcong.org/2026-06-30-why-i-stopped-arguing-with-people.html](https://wangcong.org/2026-06-30-why-i-stopped-arguing-with-people.html)

这篇文章由一位软件工程师撰写，探讨了为什么通过争论来追求技术上的正确性往往是一种失败的策略。作者反思了其职业生涯的转变：从执着于“赢得”辩论，转向优先考虑个人成长和有效沟通。

核心前提是：**大多数争论关乎自我，而非观点。**由于许多人将自己的观点与自我认同挂钩，挑战其观点会被视为人身攻击，从而引发防御性抵触，而非理性的反思。鉴于人类本质上是情感动物，往往只利用逻辑来为既有感受辩护，因此即使是无懈可击的论点也极少能改变他人的想法。

核心要点包括：

*   **正确的代价：** 赢得一场争论往往会制造出一个“输家”，从而损害人际关系和社会地位。
*   **不请自来的建议是无效的：** 人们很少通过被纠正来学习，而是从后果中学习。作者建议等待“门从内部开启”——仅在对方明确请求时提供帮助，因为只有那时对方才会放下戒备。
*   **从差异中获益：** 与其试图说服怀疑者，不如将分歧视为“护城河”。在商业和创新领域，坚持他人所排斥的信念是一种竞争优势；与其争论，不如将想法付诸实践，让现实证明其价值。
*   **专注于改变自我：** 你无法改变他人，只能改变自己。通过保持谦虚、寻求反馈而非追求胜利，你可以避开阻碍成长的自我意识。

最后，作者总结道，放下赢得争论的执念会带来平静，并让人能够专注于唯一真正能提升的对象：自己。
