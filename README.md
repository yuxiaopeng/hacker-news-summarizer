## 1. QuadRF可以穿墙发现无人机并探测WiFi信号

**原文标题**: QuadRF can spot drones and see WiFi through my wall

**原文链接**: [https://www.jeffgeerling.com/blog/2026/quadrf-can-spot-drones-and-see-wifi-through-my-wall/](https://www.jeffgeerling.com/blog/2026/quadrf-can-spot-drones-and-see-wifi-through-my-wall/)

QuadRF 是一款基于 Raspberry Pi 5 和 FPGA 构建的手持式开源相控阵无线电系统。该设备由前 SpaceX 工程师 Martin McCormick 开发（他曾参与 Starlink “Dishy”终端的研发），为开源社区带来了先进的波束成形和信号处理能力。

QuadRF 的一项主要技术创新是利用了 Raspberry Pi 5 的 MIPI（摄像头/显示器）通道进行数据传输。通过复用这些通道，该设备能够以超过 5 Gbps 的速率传输 I/Q 数据，且延迟低于传统的 USB 连接。该工具运行在 4.9–6 GHz 频段，允许用户追踪飞行中的无人机并“透视”墙壁看到 WiFi 信号。它还包含一个自定义的增强现实（AR）可视化器，可在移动界面上将射频频率呈现为彩色的“色块”。

该硬件专为可扩展性而设计；虽然手持单元旨在用于本地 SDR 应用，但多个模块可以串联在一起。这种模块化设计是一个更大项目的一部分，旨在进行地-月-地（EME）无线电实验和射电天文学研究，其等效全向辐射功率（EIRP）可达惊人的 1.15 MW。

尽管目前的工程原型采用了尚不成熟的 VNC 用户界面和 3D 打印外壳，但作者对其性能以及远距离捕获 DJI Mini Pro 4 无人机的能力印象深刻。QuadRF 目前已在 Crowd Supply 平台上开启预售，价格为 499 美元，未来的量产单元将采用注塑成型外壳。该项目最终证明，曾经仅限政府使用的复杂射频监测与分析工具，现在普通爱好者也能触手可及。

## 2. 蜗牛牙齿超越蜘蛛丝，成为自然界最强韧的材料 (2015)

**原文标题**: Snails' Teeth Beats Spider Silk as Nature's Strongest Material (2015)

**原文链接**: [https://www.smithsonianmag.com/smart-news/spider-silk-loses-top-spot-natures-strongest-material-snails-teeth-180954346/](https://www.smithsonianmag.com/smart-news/spider-silk-loses-top-spot-natures-strongest-material-snails-teeth-180954346/)

根据发表在《英国皇家学会界面杂志》上的研究，一种被称为“帽贝”的海蜗牛牙齿已被确认为地球上最强的天然材料。这一发现取代了此前的纪录保持者——蜘蛛丝，成为生物强度的新标杆。

帽贝利用一种长满牙齿、被称为“齿舌”的器官来磨碎岩石以获取食物。科学家发现，这些牙齿由包裹在蛋白质基质中的含铁针铁矿纳米纤维组成。这种独特的结构提供了极高的抗拉强度（即抵抗拉断的能力），约为大多数蜘蛛丝的五倍。该材料可与高质量碳纤维相媲美，并能承受通常使碳转化为钻石的巨大压力。为了形象地说明这一点，研究首席作者阿萨·巴伯（Asa Barber）指出，这种强度相当于一根意大利面条能吊起3300磅重的白糖。

虽然这种材料尚未超越人造石墨烯，但其性能优于凯夫拉纤维，并能与先进的合成纤维并驾齐驱。文中还区分了抗拉强度与硬度，指出尽管蓝丝黛尔石和纤锌矿型氮化硼等稀有材料比钻石更硬（即更耐刮擦），但帽贝牙齿代表了自然界抗拉伸破坏能力的最高水平。

这些发现引起了工程界的广泛关注。通过研究帽贝牙齿的微观结构，研究人员希望开发出仿生材料，并将其应用于高性能结构、机械设备以及下一代制造业。
