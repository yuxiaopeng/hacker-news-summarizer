## 3. 首例用于修复胎儿脊柱裂的宫内干细胞疗法证实安全

**原文标题**: First-ever in-utero stem cell therapy for fetal spina bifida repair is safe

**原文链接**: [https://health.ucdavis.edu/news/headlines/first-ever-in-utero-stem-cell-therapy-for-fetal-spina-bifida-repair-is-safe-study-finds/2026/02](https://health.ucdavis.edu/news/headlines/first-ever-in-utero-stem-cell-therapy-for-fetal-spina-bifida-repair-is-safe-study-finds/2026/02)

加州大学戴维斯分校健康中心（UC Davis Health）的研究人员已成功完成“CuRe试验”的第一阶段，这标志着全球首例用于修复脊柱裂（脊髓脊膜膨出）的宫内干细胞疗法取得重大进展。发表在《柳叶刀》上的研究结果证实，这一突破性手术对母亲和胎儿均具有安全性。

**手术过程**
该治疗涉及一项特殊的胎儿手术，外科医生将含有由人类胎盘源性干细胞制成的特殊补片直接覆盖在胎儿暴露的脊髓上。这种方法旨在超越标准的外科修复手术，利用干细胞的再生能力保护发育中的脊髓免受进一步损伤，并在出生前促进组织愈合。

**关键发现**
第一阶段对六名婴儿从手术到出生进行了监测。关键结果包括：
*   **安全性：** 未发现感染、脑脊液泄漏或异常组织生长/肿瘤。
*   **手术成功：** 所有伤口均完全愈合，且核磁共振（MRI）扫描显示，所有婴儿的后脑疝（手术成功的主要指标）均得到逆转。
*   **减少并发症：** 所有婴儿在出院前均无需通过分流术治疗脑积水。

**意义与后续步骤**
美国每年约有2,000名儿童受到脊柱裂的影响，这往往会导致终身的行动能力和认知障碍。虽然标准的胎儿手术已经改善了预后，但许多儿童仍面临巨大挑战。这项由加州再生医学研究所（CIRM）资助的疗法旨在进一步提高患者的生活质量和身体机能。

在获得美国食品药品监督管理局（FDA）批准后，该试验已进入1/2a阶段，将招募多达35名患者。研究人员将对这些儿童进行随访直至六岁，以评估其长期安全性以及运动、肠道和膀胱功能的改善情况。

## 4. /e/OS 是一个完整且完全“去谷歌化”的移动生态系统。

**原文标题**: /e/OS is a complete, fully “deGoogled” mobile ecosystem

**原文链接**: [https://e.foundation/e-os/](https://e.foundation/e-os/)

**/e/OS** 是一个以隐私为核心的开源移动生态系统，旨在提供完全“去谷歌化”的安卓体验。通过使用隐私友好的替代方案（例如用于核心服务的 microG 和用于搜索的 Murena Find）取代谷歌的核心服务，该操作系统确保个人数据不会被谷歌采集。其隐私标准已获得爱丁堡大学和都柏林三一学院研究人员的认可。

**主要特性：**
*   **隐私与应用管理：** /e/OS 在兼容标准安卓应用的同时，内置了“App Lounge”应用商店，可根据追踪器和权限对应用进行安全评分。“高级隐私”组件允许用户屏蔽追踪器、隐藏 IP 地址并伪造地理位置。
*   **Murena 工作空间：** 该生态系统包含一个云端套件 (murena.io)，提供电子邮件、1GB 免费存储空间及办公工具。作为 Google Workspace 或 Office 365 的私密替代方案，它为敏感文档提供端到端加密。

**获取途径：**
用户可以通过购买预装好的 **Murena 智能手机**，或使用简化的“Easy Installer”工具在自己的兼容设备上安装该系统。对于高级用户，可通过 GitLab 直接下载镜像和源代码。该项目拥有详尽的文档、社区论坛和专门的 Telegram 支持频道。
